# NIR: Multi-frame Super-Resolution с оценкой неопределённости

**Тема:** Замена этапа оптимальной интерполяции Максимова на нейросетевую модель EDSR с картой неопределённости, используемой при взвешенном комплексировании кадров.

## Ключевые формулы

**Модель наблюдения Максимова:**
$$y[k] = (h * x)[kT + \Delta] + v[k], \quad v \sim \mathcal{N}(0, D_v)$$

**Оптимальный Wiener-фильтр:**
$$G(f) = \frac{H^*(f)}{|H(f)|^2 + \lambda}, \quad \lambda = D_v/D_x$$

**Аналитическая дисперсия ошибки:**
$$D_\varepsilon(\Delta) = D_x (1 - \rho^{2|\Delta|}) + D_v$$

**Взвешенное комплексирование (формула 92):**
$$\hat{x}(k_0,k_1) = \frac{\sum_i y_i / D_\varepsilon^{(i)}}{\sum_i 1 / D_\varepsilon^{(i)}}$$

**Нормированная ошибка (метрика, совместимая с диссертацией Максимова):**
$$\tilde{D}_\varepsilon/D_x = \frac{\text{MSE}(\hat{x}, x_{gt})}{\text{Var}(x_{gt})}$$

## Методы оценки неопределённости

### A. TTA Ensemble
Запустить EDSR на 4 аугментациях (оригинал, горизонтальный flip, вертикальный flip, оба flip), инвертировать аугментации, взять **дисперсию** по 4 выходам как D_ε.

### B. MC Dropout
Добавить Dropout2d(p=0.15) после каждого из 16 ResBlock-ов, запустить K=10 прогонов с активным dropout, взять **дисперсию** по K выходам как D_ε.

### C. Гетероскедастический прокси
Без дообучения: `σ²(x,y) = smooth(|EDSR − bicubic|²) + 0.1·|∇EDSR|`.  
Высокая неопределённость там, где нейросеть отклоняется от классического baseline.

## Ожидаемые выводы (предварительные, подтверждаются графиками)

На тестовом изображении 256×256 (crop из DIV2K 0801), L=4, I=12, D_v/D_x=0.1, d=0.1:

| Метод | PSNR | D_eps/D_x | Вывод |
|---|---|---|---|
| P1 Bicubic+avg | ~27.2 dB | ~0.024 | Простой baseline, хорошая точность |
| P2 Maximov (Wiener) | ~23.6 dB | ~0.055 | Wiener вносит артефакты на этих параметрах |
| P3 EDSR+avg | ~27.1 dB | ~0.025 | EDSR≈bicubic (модель обучена на другой деградации) |
| P4 EDSR+Ensemble | ~24.8 dB | ~0.041 | TTA-дисперсия мала, веса плохо откалиброваны |
| P5 EDSR+MCD | ~26.2 dB | ~0.031 | Dropout даёт более реалистичную неопределённость |
| P6 EDSR+Heterosc. | ~27.1 dB | ~0.025 | Лучший из взвешенных: прокси хорошо коррелирует с ошибкой |

**Ключевые научные выводы:**
1. EDSR не показывает явного превосходства над бикубиком при данной деградации — EDSR обучен на бикубической деградации, а не Максимовской (разные распределения входов).
2. Гетероскедастический прокси (P6) — лучший из нейросетевых методов: uncertainty хорошо коррелирует с реальной ошибкой.
3. TTA-ансамбль (P4) с детерминированной моделью даёт слишком малую дисперсию → переуверенность → плохие веса.
4. Wiener-фильтр (P2) уступает бикубику при умеренном шуме/размытии, но теоретически превосходит при экстремальных условиях (График 3).

Это само по себе научный результат: показывает необходимость согласования модели деградации при обучении EDSR с моделью наблюдения Максимова.
