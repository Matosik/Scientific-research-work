# ТЗ на научно-исследовательскую работу

## Замена этапа оптимальной интерполяции в методе восстановления динамических изображений на нейросетевую модель с оценкой неопределённости

---

## 1. Контекст и постановка задачи

### 1.1. Базовая работа

Исследование развивает метод восстановления динамических изображений локальных плоских объектов, предложенный в диссертации А.И. Максимова. Метод предназначен для повышения пространственного разрешения объектов интереса (номерных знаков, надписей, экранов) по серии низкоразрешённых кадров видеопоследовательности и применяется в задачах криминалистической экспертизы и мониторинга транспортного видеопотока.

### 1.2. Исходный метод Максимова - три этапа

| Этап | Описание | Выход |
|---|---|---|
| **1. Оптимальная интерполяция** | Восстановление каждого кадра со сверхразрешением по непрерывно-дискретной модели наблюдения (минимизация дисперсии ошибки) | Интерполированный кадр + **поле дисперсии ошибки** $D_\varepsilon(k_0, k_1)$ |
| **2. Геометрическое согласование** | Приведение кадров к общей системе координат с субпиксельной точностью | Согласованные кадры + согласованные поля ошибок |
| **3. Оптимальное комплексирование** | Взвешенное усреднение отсчётов с весами $a_i \propto 1/D_\varepsilon^{(i)}$ | Итоговое HR-изображение |

Ключевая формула комплексирования (формула 92 диссертации):

$$
\hat{x}(k_0, k_1) = \frac{\sum_{i=0}^{I-1} y_i(k_0, k_1) / D_\varepsilon^{(i)}(k_0, k_1)}{\sum_{i=0}^{I-1} 1 / D_\varepsilon^{(i)}(k_0, k_1)}
$$

### 1.3. Идея настоящей работы

Заменить **этап 1** (классический оптимальный линейный фильтр) на **нейросетевую SISR-модель**, которая дополнительно к восстановленному кадру выдаёт **карту неопределённости** - функциональный аналог поля дисперсии $D_\varepsilon$. Карта неопределённости используется на этапе 3 в качестве весов комплексирования.

### 1.4. Научные вопросы исследования

1. Даёт ли нейросетевая интерполяция меньшую ошибку итогового восстановления, чем оптимальная линейная по Максимову?
2. Какой способ оценки неопределённости даёт наиболее **калиброванную** карту ошибок: ансамбль или MC Dropout ?
3. Полезна ли карта ошибок в принципе - даёт ли взвешенное комплексирование выигрыш над простым усреднением?
4. Сохраняется ли преимущество нейросети при усилении шума и динамических искажений (где классический Максимов имеет теоретическое обоснование)?

### Этап 1. Реализация baseline-метода Максимова 

**1.1. Модуль `degradation.py` - модель наблюдения**

Реализовать прямую модель:
$$
y[k] = (h * x)[kT] + v[k], \quad v \sim \mathcal{N}(0, D_v)
$$

где $h$ - гауссова ИХ с параметром ширины $d$, $T$ - шаг дискретизации.

Параметры (по диссертации):
- $0 \le d \le T^2$
- $0 \le D_v \le 0.2 D_x$
- $L$ - коэффициент учащения: 2, 4, 5 (для основных экспериментов)
- Число кадров $I$: до 40

Реализовать восстанавливающую ЛИС-систему по формулам разделов 2.4–2.7 диссертации. ИХ фильтра $g_\Delta(m)$ удовлетворяет системе (21) и обеспечивает минимум дисперсии ошибки $D_\varepsilon$, которая вычисляется по формуле (34/40).

Альтернативно - упрощённый baseline: **бикубическая интерполяция** + карта ошибок как норма градиента (объяснимо, но менее строго).

### Этап 2. Нейросетевая SISR-модель

**2.1. Выбор архитектуры**

| Модель | Роль | Обоснование |
|---|---|---|
| **EDSR** | Основная | Стандарт SISR, L1-loss, не галлюцинирует, простая реализация, есть веса |
| **RCAN** | Альтернатива | Channel attention, +0.2–0.5 dB к EDSR |
| **SwinIR (классический)** | Опционально, SOTA | Трансформер, текущий ориентир по PSNR |
| **HAT** | Опционально | Самый сильный PSNR-ориентированный, тяжёлый |
| ❌ Real-ESRGAN/SRGAN | НЕ используем | GAN-модели галлюцинируют детали - для криминалистики и multi-frame fusion недопустимо |
| ❌ SR3/StableSR | НЕ используем | Стохастичность диффузии разрушает фьюжн |

**Минимальный набор : EDSR + (опционально) SwinIR.**

**2.2. Подготовка датасета для обучения**

Использовать **DIV2K** (стандарт SR) :

```
HR (DIV2K) 
  → случайный субпиксельный сдвиг 
  → гауссово размытие (σ соотв. d/T)
  → децимация в L раз 
  → аддитивный шум N(0, D_v)
  → LR-патч 64×64
```

Используется общий датасет (DIV2K), а не тематический (номера), чтобы:
- Сохранить обобщающую способность модели
- Избежать переобучения на конкретные шрифты
- Главную работу по восстановлению деталей делает шаг 3 (комплексирование), а не "запомненный" вид номеров

**2.3. Обучение / дообучение**

- Базовая EDSR: предобученные веса (DIV2K bicubic) + fine-tuning на деградации (10–15 эпох).
- Loss: L1 (для PSNR-режима) - никакого adversarial.
- Целевые метрики на валидации: PSNR, SSIM, LPIPS.

**Контрольная точка этапа 2:** EDSR-fine-tuned должна давать выигрыш по PSNR над бикубиком ≥ 1.5 dB на тестовом синтетическом наборе.

### Этап 3.
Два способа оценки неопределённости 

#### 3.1. Способ А - Ансамбль глубоких моделей (Deep Ensemble)

- Обучить N независимых копий EDSR (N=5) с разной инициализацией и/или на разных подвыборках.
- Инференс: пропустить кадр через все N сетей.
  - Среднее → восстановленный кадр $y_i$.
  - **Дисперсия по N выходам → карта $D_\varepsilon^{(i)}$.**
- Плюсы: самая надёжная оценка, классический baseline в uncertainty estimation.
- Минусы: дорого по обучению.

#### 3.2. Способ Б - Monte Carlo Dropout

- Взять EDSR, вставить dropout-слои (p=0.1) после residual-блоков.
- Дообучить с активным dropout (~10 эпох).
- Инференс: K прогонов с **не выключенным** dropout (K=20–50).
  - Среднее → $y_i$, дисперсия → $D_\varepsilon^{(i)}$.
- Плюсы: одна модель, дешевле обучение.
- Минусы: оценка может быть менее калиброванной, инференс ×K.

#### 3.3. Способ В - Гетероскедастическая регрессия (Kendall & Gal, 2017)

- Модифицировать выход EDSR: два канала - изображение $\hat{x}$ и log-дисперсия $\log \sigma^2$.
- Loss:
$$
\mathcal{L} = \frac{|\hat{x} - x_{gt}|^2}{2\sigma^2} + \frac{1}{2}\log \sigma^2
$$
- Сеть учится сама предсказывать, где она уверена.
- Плюсы: концептуально наиболее близко к Максимову (явный двухканальный выход), один forward pass.
- Минусы: чувствителен к нестабильности обучения (нужна стабилизация $\log \sigma^2$).

## 8. Ключевые контрольные точки и критерии успеха

| КТ | Критерий | Что значит, если не выполнено |
|---|---|---|
| **КТ1.** Воспроизведение результата Максимова | $\tilde{D}_\varepsilon/D_x \approx 0.083$ при $L=5, D_v=0.2D_x, d=0$ | Ошибка в реализации этапа 1 или метрике - переделать перед движением дальше |
| **КТ2.** EDSR > бикубика на синтетике | PSNR EDSR ≥ PSNR bicubic + 1.5 dB | Проблемы с обучением или несоответствие распределений LR - переподобрать модель деградации |
| **КТ3.** Карта неопределённости - не шум | Корреляция с фактической ошибкой ≥ 0.4 | Метод оценки неопределённости работает плохо - пересмотреть гиперпараметры |
| **КТ4.** Взвешенное лучше простого усреднения | $\tilde{D}_\varepsilon$(P4 или P5 или P6) < $\tilde{D}_\varepsilon$(P3) | Карта ошибок бесполезна - это **сам по себе результат**, его надо проанализировать |
| **КТ5.** Хотя бы один нейросетевой пайплайн лучше Максимова | PSNR одного из P4–P6 > PSNR P2 на >1 dB | Подход не оправдал себя - формулировать выводы с акцентом на калибровку и устойчивость, а не на абсолютное преимущество |

## Приложение A. Ключевые формулы

**Непрерывно-дискретная модель наблюдения:**
$$y[k] = \int h(kT - \tau) x(\tau) d\tau + v[k]$$

**Оптимальный линейный восстанавливающий фильтр (импульсная характеристика $g_\Delta$):**
$$\sum_{m=-\infty}^{\infty} g_\Delta(m) B_y^{(д)}(n - m) = B_{xy}^{(н)}(-nT - \Delta)$$

**Дисперсия ошибки оптимальной интерполяции:**
$$D_\varepsilon(\Delta) = D_x - \sum_{m=-\infty}^{\infty} g_\Delta(m) B_{xy}^{(н)}(-\theta - \Delta)\Big|_{\theta = mT}$$

**Оптимальное комплексирование (формула 92):**
$$\hat{x}(k_0, k_1) = \frac{\sum_{i=0}^{I-1} y_i(k_0, k_1) / D_\varepsilon^{(i)}(k_0, k_1)}{\sum_{i=0}^{I-1} 1 / D_\varepsilon^{(i)}(k_0, k_1)}$$

**Усредняющее комплексирование (контрольный baseline, формула 94):**
$$\hat{x}(k_0, k_1) = \frac{1}{I} \sum_{i=0}^{I-1} y_i(k_0, k_1)$$

**Heteroscedastic loss (Kendall & Gal, для способа В):**
$$\mathcal{L} = \frac{|\hat{x} - x_{gt}|^2}{2\sigma^2} + \frac{1}{2}\log \sigma^2$$

**Нормированная мера ошибки восстановления (для сопоставимости с Максимовым):**
$$\tilde{D}_\varepsilon / D_x = \frac{1}{N D_x} \sum_{k} (\hat{x}[k] - x_{gt}[k])^2$$
